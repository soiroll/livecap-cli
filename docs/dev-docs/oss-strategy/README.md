## 🤔 よくある質問

### Q: どのドキュメントから読めばいい？

A: [00-overview.md](00-overview.md)から始めてください。全体像が理解できます。

### Q: Steam確認なしに進めていい？

A: **絶対にダメです。** Steam規約違反のリスクがあります。必ず事前確認してください。

### Q: 商用ライセンスは必須？

A: いいえ。完全GPL（商用ライセンスなし）も選択肢ですが、「ただ乗り防止」が目的なら推奨します。

### Q: 初期投資はいくら必要？

A: **ゼロ円で可能**です。弁護士レビュー（$500-$2,000）はオプションです。

### Q: 実装にどれくらい時間がかかる？

A: 最短4週間、推奨8週間、慎重なら3ヶ月です。週数時間の作業を想定しています。

### Q: 既存購入者から返金要求される？

A: Option 1（プレミアム化）なら可能性は低いです。丁寧な説明と感謝の表明が重要です。
